# 📖 XRL 指令字典 v1.0

## 一、数据操作指令

### xr — 写入单元格
| 项目 | 内容 |
|------|------|
| **格式** | `xr/地址/内容` |
| **说明** | 向指定单元格写入内容 |
| **参数** | 地址：单元格引用（如 A1）<br>内容：任意文本 |
| **示例** | `xr/A1/姓名` |
| **备注** | 内容中可包含 `/`，引擎会自动处理 |

---

### sum — 求和
| 项目 | 内容 |
|------|------|
| **格式** | `sum/范围/目标` |
| **说明** | 计算范围内所有数字的总和 |
| **参数** | 范围：单元格范围（如 B2:B5）<br>目标：结果存放的单元格 |
| **示例** | `sum/B2:B5/B6` |
| **备注** | 自动忽略非数字单元格，范围必须包含 `:` |

---

### avg — 平均值
| 项目 | 内容 |
|------|------|
| **格式** | `avg/范围/目标` |
| **说明** | 计算范围内所有数字的平均值 |
| **参数** | 范围：单元格范围（如 B2:B5）<br>目标：结果存放的单元格 |
| **示例** | `avg/B2:B5/B7` |
| **备注** | 结果保留2位小数 |

---

### sub — 减法
| 项目 | 内容 |
|------|------|
| **格式** | `sub/被减数/减数/目标` |
| **说明** | 计算 被减数 - 减数 |
| **参数** | 被减数：单元格引用<br>减数：单元格引用<br>目标：结果存放的单元格 |
| **示例** | `sub/C2/B2/D2` |
| **备注** | D2 = C2 - B2 |

---

### mul — 乘法
| 项目 | 内容 |
|------|------|
| **格式** | `mul/乘数A/乘数B/目标` |
| **说明** | 计算 乘数A × 乘数B |
| **参数** | 乘数A：单元格引用<br>乘数B：单元格引用<br>目标：结果存放的单元格 |
| **示例** | `mul/B2/C2/D2` |
| **备注** | D2 = B2 × C2 |

---

### div — 除法
| 项目 | 内容 |
|------|------|
| **格式** | `div/被除数/除数/目标` |
| **说明** | 计算 被除数 ÷ 除数 |
| **参数** | 被除数：单元格引用<br>除数：单元格引用<br>目标：结果存放的单元格 |
| **示例** | `div/B2/C2/D2` |
| **备注** | 除数为0时显示 `#DIV/0!`，结果保留4位小数 |

---

### max — 最大值
| 项目 | 内容 |
|------|------|
| **格式** | `max/范围/目标` |
| **说明** | 找出范围内的最大值 |
| **参数** | 范围：单元格范围<br>目标：结果存放的单元格 |
| **示例** | `max/B2:B10/C1` |
| **备注** | 无有效数字时结果为 0 |

---

### min — 最小值
| 项目 | 内容 |
|------|------|
| **格式** | `min/范围/目标` |
| **说明** | 找出范围内的最小值 |
| **参数** | 范围：单元格范围<br>目标：结果存放的单元格 |
| **示例** | `min/B2:B10/C1` |
| **备注** | 无有效数字时结果为 0 |

---

### count — 计数
| 项目 | 内容 |
|------|------|
| **格式** | `count/范围/目标` |
| **说明** | 统计范围内非空单元格数量 |
| **参数** | 范围：单元格范围<br>目标：结果存放的单元格 |
| **示例** | `count/A2:A20/B1` |
| **备注** | 空字符串不算非空 |

---

### if — 条件判断
| 项目 | 内容 |
|------|------|
| **格式** | `if/条件/真值/假值/目标` |
| **说明** | 根据条件选择输出值 |
| **参数** | 条件：条件表达式<br>真值：条件成立时的值<br>假值：条件不成立时的值<br>目标：结果存放的单元格 |
| **示例** | `if/A1>=60/及格/不及格/B1` |
| **备注** | 条件格式：`单元格 运算符 值` |

---

## 二、格式操作指令

### bg — 背景色
| 项目 | 内容 |
|------|------|
| **格式** | `bg/目标/颜色` |
| **说明** | 设置单元格或行的背景色 |
| **参数** | 目标：HEADER / ROW_N / 单元格地址<br>颜色：颜色名称或 #RRGGBB |
| **示例1** | `bg/HEADER/蓝色背景` |
| **示例2** | `bg/ROW_5/合计背景` |
| **示例3** | `bg/B2/红色背景` |
| **示例4** | `bg/A1/#FFE0E0` |

---

### ct — 加粗
| 项目 | 内容 |
|------|------|
| **格式** | `ct/地址` |
| **说明** | 将单元格文字加粗 |
| **参数** | 地址：单元格引用 |
| **示例** | `ct/A5` |
| **备注** | 常用于合计行、表头 |

---

### it — 斜体
| 项目 | 内容 |
|------|------|
| **格式** | `it/地址` |
| **说明** | 将单元格文字设为斜体 |
| **参数** | 地址：单元格引用 |
| **示例** | `it/A1` |

---

### ul — 下划线
| 项目 | 内容 |
|------|------|
| **格式** | `ul/地址` |
| **说明** | 给单元格文字添加下划线 |
| **参数** | 地址：单元格引用 |
| **示例** | `ul/A1` |

---

### fc — 字体颜色
| 项目 | 内容 |
|------|------|
| **格式** | `fc/地址/颜色` |
| **说明** | 设置单元格文字颜色 |
| **参数** | 地址：单元格引用<br>颜色：颜色名称或 #RRGGBB |
| **示例1** | `fc/A1/红色` |
| **示例2** | `fc/B2/#FF0000` |

---

### fs — 字号
| 项目 | 内容 |
|------|------|
| **格式** | `fs/地址/字号` |
| **说明** | 设置单元格字号 |
| **参数** | 地址：单元格引用<br>字号：6-72 之间的整数 |
| **示例** | `fs/A1/16` |
| **备注** | 范围 6-72，超出报错 |

---

### align — 对齐方式
| 项目 | 内容 |
|------|------|
| **格式** | `align/地址/对齐方式` |
| **说明** | 设置单元格水平对齐 |
| **参数** | 地址：单元格引用<br>对齐：left / center / right |
| **示例** | `align/A1/center` |
| **备注** | 只能为 left、center、right 之一 |

---

### highlight — 条件高亮单元格
| 项目 | 内容 |
|------|------|
| **格式** | `highlight/范围/条件/颜色` |
| **说明** | 范围内满足条件的单元格高亮 |
| **参数** | 范围：单元格范围<br>条件：条件表达式<br>颜色：颜色名称（可选，默认红色背景） |
| **示例1** | `highlight/B2:D10/<60/红色背景` |
| **示例2** | `highlight/B2:D10/>=90/绿色背景` |
| **示例3** | `highlight/C2:C20/<0/红色背景` |
| **备注** | 条件支持 `<60`、`>=90`、`==0` 等简化格式 |

---

### highlightrow — 条件高亮整行
| 项目 | 内容 |
|------|------|
| **格式** | `highlightrow/范围/条件/颜色` |
| **说明** | 范围内满足条件的单元格所在整行高亮 |
| **参数** | 范围：单元格范围<br>条件：条件表达式<br>颜色：颜色名称（可选，默认红色背景） |
| **示例** | `highlightrow/A2:E10/总分<200/黄色背景` |

---

## 三、结构操作指令

### ar — 添加行
| 项目 | 内容 |
|------|------|
| **格式** | `ar/数量` |
| **说明** | 在表格末尾添加行 |
| **参数** | 数量：1-50，默认1 |
| **示例** | `ar/5` |
| **备注** | 总数不超过500行 |

---

### ac — 添加列
| 项目 | 内容 |
|------|------|
| **格式** | `ac/数量` |
| **说明** | 在表格右侧添加列 |
| **参数** | 数量：1-20，默认1 |
| **示例** | `ac/3` |
| **备注** | 总数不超过50列 |

---

### cl — 清除单元格
| 项目 | 内容 |
|------|------|
| **格式** | `cl/地址` |
| **说明** | 清除指定单元格的内容和格式 |
| **参数** | 地址：单元格引用 |
| **示例** | `cl/A1` |

---

### mv — 移动单元格
| 项目 | 内容 |
|------|------|
| **格式** | `mv/源地址/目标地址` |
| **说明** | 将源单元格的内容和格式移动到目标单元格 |
| **参数** | 源地址：单元格引用<br>目标地址：单元格引用 |
| **示例** | `mv/A1/B2` |
| **备注** | 移动后源单元格清空 |

## 五、条件表达式语法

| 格式 | 说明 | 示例 |
|------|------|------|
| `运算符 值` | 与单元格数值比较 | `<60` `>=90` `==100` `!=0` |
| `单元格 运算符 值` | 指定单元格比较 | `A1>=60` `B2<100` |

**支持的运算符：** `>` `<` `>=` `<=` `==` `!=`

## 八、注释

以 `#` 开头的行被视为注释，不会执行：

```

这是注释，下面是指令

xr/A1/姓名
xr/B1/分数

```

## 九、执行规则

| 规则 | 说明 |
|------|------|
| 执行顺序 | 从上到下，先执行数据指令，再执行高亮指令 |
| 空行处理 | 自动忽略空行 |
| 错误处理 | 单条指令失败不影响后续指令执行 |
| 大小写 | 指令类型不区分大小写（`SUM` = `sum`） |
| 地址大小写 | 列字母不区分大小写（`a1` = `A1`） |
```
